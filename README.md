## 🚀 クイックスタート

### 1. フォルダを整理（初回のみ）

```bash
# このファイルがあるフォルダで実行
reorganize.bat
```

### 2. 自動化ツールのセットアップ

```bash
cd 02_自動化ツール
setup.bat
```

### 3. ツールを起動

```bash
# 半自動モード（推奨・低コスト）
manual.bat

# 完全自動モード
run.bat configs\ピンクノイズ_8時間.yaml
```

## 💰 2つのモード

### 🎨 半自動モード（推奨）

**起動**: `02_自動化ツール\manual.bat`

**特徴**:
- Mubert APIを使わない（月$14のみ）
- GUIで簡単操作
- チェックリスト表示
- テンプレート選択→プロンプト自動表示
- ロング/ショート両対応

**手順**:
1. テンプレート選択
2. プロンプトコピー
3. Mubertサイトで手動生成
4. 音源をダウンロード
5. GUIで動画化・アップロード

**費用**: $14/月  
**時間**: 60〜90分/本

---

### 🤖 完全自動モード

**起動**: `02_自動化ツール\run.bat`

**特徴**:
- すべて自動実行
- 設定ファイルのみ準備
- 放置でOK

**手順**:
1. 設定ファイル作成
2. run.bat実行
3. 待つだけ

**費用**: $34〜$54/月  
**時間**: 90〜200分/本（放置可能）
