# 🚀 AI音楽BGMチャンネル - スタートガイド

このプロジェクトで睡眠・作業用BGM動画を作成し、YouTube収益化を目指します。

---

```

├── 01_テンプレート/            ← 21種類の動画テンプレート
```

---

## ⚡ 3ステップで開始

### ステップ1：フォルダを整理（初回のみ・1分）

```bash
# このファイルがあるフォルダで実行
reorganize.bat
```

これでファイルが整理され、上記の構成になります。

### ステップ2：自動化ツールをセットアップ（初回のみ・5分）

```bash
cd 02_自動化ツール
setup.bat
```

指示に従ってMubert APIキーを設定します。

### ステップ3：動画を作成（30〜90分）

```bash
cd 02_自動化ツール
manual.bat
```

GUIが開きます。詳細は `02_自動化ツール/QUICKSTART.md` を参照。
