## フェーズ別ロードマップ

### フェーズ1：チャンネル基盤構築（1〜2ヶ月目）

**目標：登録者1,000人・総再生時間4,000時間達成**

#### 長尺動画戦略
- **投稿頻度**：週3〜4本
- **優先テーマ**（既存21テンプレートから）：
  1. ピンクノイズ/ブラウンノイズ（検索需要大）
  2. 雨音/波音（季節を問わず人気）
  3. ピアノ/アンビエント（作業用で需要高）
  4. 焚き火/森の音（癒し系で差別化）

- **長さ別設定**：
  - **睡眠用**：8〜10時間（夜通し再生狙い）
  - **作業用**：3〜5時間（在宅ワーク1セッション）
  - **短尺版**：25分（ポモドーロタイマー需要）

#### ショート動画導入
- **投稿頻度**：毎日1〜2本
- **役割**：長尺動画への導線＋新規視聴者獲得
- **制作方法**：長尺動画から30〜60秒切り出し＋独自ショート

**目標：月間広告収益5万円以上**

- **作業用3時間プレイリスト作成**

  - 午後の部（13:00〜18:00想定）
  - 深夜の部（22:00〜01:00想定）

#### ショート動画の戦略的活用

  - 「10秒で眠くなる音」系フック
  - 「作業効率2倍BGM」効果訴求

  - トレンド音源の睡眠版アレンジ

- **シリーズ化**：
  - 「1分で眠くなるシリーズ」Day 1, 2, 3...

  - 「○○しながら聞くと効果倍増」Tips系

## ショート動画戦略

### ショート動画テンプレート

#### パターンA：フック型（インパクト重視）
```
[0〜3秒] 衝撃的テキスト
「10秒で眠くなる禁断の音」

[3〜30秒] BGMスタート＋ビジュアル
波形や抽象映像

[30〜50秒] 説明
「ピンクノイズは脳波を...」

[50〜60秒] CTA
「フル版は概要欄↓」
```

#### パターンB：比較型
```
[0〜5秒] 「どっちが眠れる？」

[5〜25秒] 画面分割
左：普通の雨音
右：睡眠特化雨音

[25〜50秒] 違いを解説
波形表示など

[50〜60秒] 投票呼びかけ
「コメントで教えて！」
```

#### パターンC：Tips型
```
[0〜5秒] 「作業効率3倍の秘訣」

[5〜35秒] 3つのポイント
高速テロップで箇条書き

[35〜55秒] 「このBGMを流すだけ」
音源プレビュー

[55〜60秒] CTA
```

### ショート動画最適化

**視覚要素**
- 9:16縦型必須
- テキストサイズ：画面の1/4以上
- 動きのある背景（波形、パーティクル）
- 1秒ごとにカット切り替え

**音響要素**
- 冒頭3秒でインパクト音
- 30秒時点で音量変化
- 最後まで音が続く設計

**テキスト最適化**
- ハッシュタグ：5〜10個
  - `#shorts #睡眠導入 #作業用bgm #集中 #勉強 #リラックス #癒し #asmr #ピンクノイズ #lo-fi`
- 概要欄：長尺動画リンク必須
- タイトル：15文字以内で完結

## 収益最大化施策

### 1. 広告配置の最適化

**8時間睡眠動画**
- プレロール：あり
- ミッドロール：30分ごと（全16箇所）
- ポストロール：あり
- 想定RPM：300〜500円

**3時間作業動画**
- プレロール：あり
- ミッドロール：20分ごと（全9箇所）
- ポストロール：あり
- 想定RPM：400〜600円

**25分短尺動画**
- プレロール：あり
- ミッドロール：なし
- ポストロール：あり
- 想定RPM：200〜300円

### 2. 視聴維持率の向上施策

**冒頭5秒**
- タイトルテキストを画面中央に大きく表示
- 「8時間」「途切れない」などキーワード強調

**1分時点**
- 軽い視覚変化（色調shift、エフェクト追加）
- 「いいね・チャンネル登録お願いします」軽く表示

**中盤（2〜6時間）**
- 20分ごとに微妙な変化（背景色の明度変更など）
- 視聴者が「まだ流れている」と認識できる程度

**終盤（残り10分）**
- 次の動画への誘導カード表示
- プレイリスト自動再生への誘導

### 3. SEO最適化

**タイトル構成**
```
【時間】【効果】【ジャンル】｜【英語】【補足】
例：【8時間】深い眠りのピンクノイズ｜Pink Noise for Deep Sleep, 朝まで途切れない
```

**説明欄最適化**
```
[最初の3行]
キーワード詰め込み＋視聴者ベネフィット

[タイムスタンプ]
0:00 導入
0:30 本編開始
7:55:00 エンディング

[関連動画リンク]
同ジャンル3本

[ハッシュタグ]
最大15個

[クレジット]
音源：Mubert (Creatorプラン)
映像：Canva Pro
```

**タグ戦略**
- 競合上位3チャンネルのタグをTubeBuddy等で分析
- 20個上限まで使い切る
- 日本語10個＋英語10個

### 4. サムネイル統一ルール

**配色ルール**
- 青系：睡眠用（ピンクノイズ、雨音など）
- 緑系：自然音（森、波など）
- オレンジ系：作業用（ピアノ、ローファイなど）
- 紫系：特殊系（シータ波、ASMRなど）

**必須要素**
- 時間表示：左上に大きく（例：8H, 3H）
- ジャンル表記：中央（例：PINK NOISE, RAIN）
- チャンネルロゴ：右下に統一配置
- エフェクト：グロー、グラデーション

## リスク管理

### 著作権対策

**事前準備**
- Mubertライセンス証明を全動画分フォルダ管理
- 説明欄に音源クレジット必須記載
  ```
  音源：Mubert (Creatorプラン / 商用利用許諾済み)
  ビジュアル：Canva Pro
  ```

**Content ID対応**
- 申請が来たら即座に異議申し立て
- ライセンス証明を添付
- 24時間以内に対応

**予防策**
- 同じ音源を複数動画で使い回さない
- 最低でも微調整（EQ、リバーブ）を加える
- 定期的にMubert利用規約の変更確認

### YouTube規約遵守

**スパム回避**
- 1日の投稿上限：3本まで（ショート含む）
- 同一音源の再利用は週1回まで
- タイトル・サムネに過度な誇張なし

**再利用コンテンツ対策**
- 各動画に固有のビジュアル要素追加
- 6時間以上の動画は3箇所以上で視覚変化
- 説明欄に各動画の独自性を明記

**コミュニティガイドライン**
- 医療効果の断定表現を避ける
  - NG：「不眠症が治る」
  - OK：「快眠をサポート」
- 子ども向けコンテンツ設定を適切に
- 暴力的・性的表現なし

### チャンネル停止リスク対策

**バックアップ**
- 全動画のローカルバックアップ保存
- サブチャンネル作成（相互バックアップ）
- 外部プラットフォームでも配信

**収益分散**
- YouTube広告のみに依存しない
- メンバーシップ導入（登録者1万人以降）
- Patreon等外部支援プラットフォーム活用
- 企業案件開拓
