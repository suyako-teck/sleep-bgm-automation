# 🌿 自然音テンプレート

リラックスと癒しの自然環境音

---

## このカテゴリのテンプレート

### 雨音
- **特徴**: 窓辺に降る優しい雨音
- **用途**: 睡眠導入、読書、リラックス
- **人気度**: ⭐⭐⭐⭐⭐ 非常に人気

### 水音
- **特徴**: 小川のせせらぎ、透明感のある流水
- **用途**: 瞑想、ヨガ、集中作業
- **人気度**: ⭐⭐⭐⭐

### 森
- **特徴**: 鳥のさえずり、風、遠くのせせらぎ
- **用途**: 自然を感じたい時、森林浴気分
- **人気度**: ⭐⭐⭐⭐

### 海辺
- **特徴**: 穏やかな波音と遠い海鳥
- **用途**: リゾート気分、リラックス
- **人気度**: ⭐⭐⭐⭐⭐

### 森の夜
- **特徴**: コオロギ、フクロウ、静かな夜風
- **用途**: 夜の睡眠導入
- **人気度**: ⭐⭐⭐

### 自然（汎用）
- **特徴**: 複数の自然音を組み合わせ
- **用途**: オールマイティ
- **人気度**: ⭐⭐⭐
