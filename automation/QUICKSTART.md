# 🚀 クイックスタート - 5分で開始

## ステップ1: セットアップ（5分）

```bash
cd automation
setup.bat
```

- 仮想環境作成
- ライブラリインストール
- .env作成

## ステップ2: YouTube認証（10分）

1. Google Cloud Consoleで `credentials.json` 取得
2. `automation/` フォルダに配置
3. `auth.bat` 実行

## ステップ3: 動画作成（60〜90分）

```bash
manual.bat
```

GUIで：
1. テンプレート選択（例：ピンクノイズ）
2. プロンプトコピー → Mubert開く
3. Mubertで10分音源生成
4. ダウンロード → 音源追加
5. 生成＋アップロード
