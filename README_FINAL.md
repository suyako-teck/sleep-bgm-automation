```

│  │  ├─ manual.bat          ⭐ 動画作成（メイン）

│  │  ├─ manual_mode_v2.py   GUIアプリ
```

## 🚀 3ステップで開始

### ステップ1: セットアップ
```bash
cd automation
setup.bat
```

### ステップ2: テスト
```bash
test.bat
```

### ステップ3: 動画作成
```bash
manual.bat
```
